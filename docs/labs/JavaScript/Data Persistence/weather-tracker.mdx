# 🌦️ World Weather Tracker

In this project, you will build a real-time Weather App. This will test your ability to use `async` and `await` to talk to an external server.

### 📅 Submission Details

- **Deadline:** Tuesday, April 21, 2026
- **Requirements:** GitHub Repository URL + Vercel Live Link
- **Late Policy:** -20 points for late submissions.

---

## 🔑 Step 1: How to get your API Key

An **API Key** is like a password that lets your code talk to the Weather Server. Follow these steps carefully:

1. **Sign Up:** Go to [OpenWeatherMap.org](https://openweathermap.org/api) and click **"Sign Up"**.
2. **Verify:** Check your email and click the verification link they sent you. **(This is important!)**
3. **Go to API Keys:** Once logged in, click on your **Username** at the top right and select **"My API Keys"**.
4. **Generate/Copy:** You will see a default key. Copy that long string of letters and numbers.
5. **Wait:** ⚠️ New keys can take **30 to 60 minutes** to activate. If you get a "401 Error," just wait a bit and try again later.

<TabItem value="js" label="script.js">

```javascript title="script.js"
// 1. Setup variables and paste your key here
const apiKey = "PASTE_YOUR_API_KEY_HERE";
const btn = document.getElementById("search-btn");
const input = document.getElementById("city-input");

// 2. The Async function (The "Brain")
async function checkWeather() {
  const city = input.value;
  // The URL address where the weather data lives
  const url = `https://api.openweathermap.org/data/2.5/weather?q=${city}&units=metric&appid=${apiKey}`;

  try {
    // AWAIT: Stop and wait for the server to answer
    const response = await fetch(url);

    // AWAIT: Stop and wait for the data to be converted to JSON
    const data = await response.json();

    if (data.cod === "404") {
      alert("City not found! Check your spelling.");
      return;
    }

    // 3. Update the HTML elements with our data
    document.getElementById("weather-info").style.display = "block";
    document.getElementById("city").innerText = data.name;
    document.getElementById("temp").innerText =
      Math.round(data.main.temp) + "°C";
    document.getElementById("desc").innerText = data.weather[0].description;
  } catch (error) {
    console.log("Something went wrong!");
  }
}

// 4. Run the function when the button is clicked
btn.addEventListener("click", checkWeather);
```
